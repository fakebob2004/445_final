\section*{Abstract}

This report documents a wristband-style ground unit that uses an ESP32, a VC-02 offline voice module,
and an SSD1306 OLED to command a five-inch quad through ExpressLRS CRSF frames.
Flight development included Betaflight telemetry trials followed by an ArduPilot/ArduCopter port for
native altitude-hold modes.
Seven spoken commands map to CRSF sequences on a RadioMaster RP2 transmitter module paired with an
RX24T receiver and a SpeedyBee F405~V5 stack.
