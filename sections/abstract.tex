\section*{Abstract}

This report presents the design and verification of a voice-controlled quadrotor drone system.
The ground unit consists of an ESP32 microcontroller paired with a VC02 offline voice
recognition module and an SSD1306 OLED display.
Seven spoken commands---takeoff, land, forward, backward, left, right, and stop---are mapped
to pre-defined RC channel sequences encoded as CRSF frames and transmitted at 50~Hz to a
RadioMaster RP2 module reflashed with ExpressLRS transmitter firmware.
The RP2 bridges the commands over a 2.4~GHz RF link to a RadioMaster RX24T receiver on the
aircraft, which forwards decoded channels to a SpeedyBee F405~V5 flight controller running
Betaflight.
Four 1860~KV brushless motors driven by an OX32 55~A ESC execute the commanded
maneuvers.
Barometric altitude telemetry is returned over the same RF link and displayed on the ground
unit's OLED in real time.
